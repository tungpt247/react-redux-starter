#Getting started

A minimal Starter Kit for React - Redux template

###What's included:

Tasks scripts using npm run

- Babel compiler ES6 to ES5

- ES6 lint, babel-lint

##Installation

Install dependencies packages

```
	npm install
```

Development version and runs a watcher

```
	npm run start
```

Runs the tests

```
	npm run test
```

Products production version

```
	npm run buid
```
